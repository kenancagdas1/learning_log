Learning Log
Learning Log is a django project.
It allows to follow and remember all the learning process. 
The user can record what he has learned for each of the numerous learning topics. 
He can easily follow along and check the order in which he has learned.
The main file is the 'manage' file.
You have to import and activate python environment to use project.
Then you have to enter right codes to command line to run the project.
